\# FUTURE\_DS\_01 — Business Sales Performance Analytics (Superstore)



Future Interns Data Science \& Analytics Task 1 (2026): analyze business sales performance using the Kaggle Superstore dataset.



\## Goal

Answer business questions such as:

\- Which products drive the most revenue and profit?

\- How do sales and profit change over time?

\- Which categories/regions are most profitable?

\- Where should the business focus to grow?



\## Tools

\- Python for data understanding, cleaning, and analysis

\- Power BI for a client-ready dashboard



\## Repository Structure

\- `data/raw/` — original dataset (not tracked by git)

\- `data/processed/` — cleaned/export-ready datasets (not tracked by git)

\- `notebooks/` — analysis notebooks

\- `powerbi/` — Power BI `.pbix` + dashboard exports

\- `reports/` — written insights and recommendations



\## Dataset

Kaggle Superstore Dataset: https://www.kaggle.com/datasets/vivek468/superstore-dataset-final



\## Deliverables

\- Cleaned dataset ready for reporting

\- Power BI dashboard (KPI + trends + breakdowns)

\- Insights + actionable recommendations
